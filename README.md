# 🍲 Nutri-Cook

Nutri-Cook is a full-stack food ordering and personalized recipe recommendation web application designed to promote healthy eating.

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 🍽 Browse Food Items
- 🛒 Add to Cart & Checkout
- 🧠 AI-Based Recipe Recommendation (future scope / optional)
- 👨‍💼 Admin Panel (Manage products & orders)

## 🛠 Tech Stack
- Frontend: React.js (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB

## 📁 Project Structure
nutri-cook/
├── nutricook-frontend/
├── nutricook-backend/
## ⚙️ Installation

### 1. Clone Repository

git clone https://github.com/chandrikachandrika23/Nutri-Cook.git

cd Nutri-Cook
### 2. Setup Backend

cd nutricook-backend
npm install
node server.js

### 3. Setup Frontend
cd ../nutricook-frontend
npm install
npm start
